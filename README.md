Este repositorio está orientado a recopilar los códigos empleados en los diferentes experimentos 
de la asignatura Física Experimental III. 

En General-codes se encuentra un código general que permite realizar ajustes, graficar datos y
realizar cálculos con grnades conjuntos de datos de manera cómoda. Por ello mismo, muchos de los
códigos presentes en las diferentes prácticas no son más que la aplicación del código general al caso concreto.
